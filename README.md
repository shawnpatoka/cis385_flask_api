**Functionality:**

 - **1. GET: /api/v1/notes** 
 Retrieve all notes
- **2. GET: /api/v1/notes/< int: id >**
Retrieve note by ID
- **3. GET: /api/v1/notes/< str: title >**
Retrieve note by Title
- **4. PUT: /api/v1/notes/< int: id >**
Updates note by ID
- **5. PUT: /api/v1/notes/< str: title >**
Updates note by Title
- **6. POST: /api/v1/notes**
Adds a new note
- **7. DELETE: /api/v1/notes/< int: id >**
Deletes a note by ID
